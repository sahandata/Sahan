# Sahan – Online Data & Airtime Selling

A static HTML dashboard for selling mobile data bundles and airtime.  
Built for small resellers / admin management.

## Pages
- `Index.html` – Home / landing page  
- `Login.html` & `Forgot-password.html` – Authentication  
- `Admin.html` – Admin dashboard  
- `Add-products.html` – Add new data/airtime products  
- `Edite-profile (1).html` – Edit user profile  
- `Order-history.html` – View past orders  
- `Leaderboard.html` – Top buyers/resellers  
- `Notification.html` – System alerts  
- `Policy.html` – Terms & privacy  

## How to use (static version)
1. Clone the repo  
2. Open any `.html` file directly in a browser  
3. For full functionality, connect a backend (Node/PHP) and database  

## Live demo (GitHub Pages)
- Go to **Settings → Pages** → Branch: `main` → Save  
- Your site will be available at `https://sahandata.github.io/Sahan/`
